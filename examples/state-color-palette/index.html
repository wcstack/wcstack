wcstack: color palette picker

state + eyedropper + clipboard + storage — pick colors from anywhere on screen, click a swatch to copy its hex, and the palette survives reloads (and syncs across tabs).

Palette ()

  • eyedropper: command.open starts the picker, the result returns as the value event token ({ sRGBHex }) — a command → event round trip. Esc cancels silently (cancelled, not error).
  • clipboard: the swatch click handler resolves the row via the wildcard path this["list.*.hex"] and emits it — emit(hex) arguments pass straight through to writeText(hex). There is deliberately no "copied" success event; failures surface on the error output.
  • storage: one two-way binding (value: palette) gives load-on-connect, save-on-assign, and cross-tab sync via the native storage event. Open this page in two tabs and pick a color.
  • The EyeDropper API needs Chromium + secure context (http://localhost is fine). Everything else — including the manual color input — works in any modern browser.
