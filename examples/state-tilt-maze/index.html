wcstack: tilt ball maze

tilt + accelerometer + timer + wakelock + state — tilt your phone to roll the ball to the flag. Even the game loop is a declarative tag.

game loop
  • Sensors need a secure context: localhost works; on a phone use HTTPS (or adb reverse).
  • iOS grants tilt via the Start button's tap (gesture-gated permission, absorbed by <wcs-tilt>).
  • Shake detection uses <wcs-accelerometer> (Chromium/Android); elsewhere the Retry button covers it.
  • Desktop: drag the board, or emulate orientation in DevTools > Sensors.
