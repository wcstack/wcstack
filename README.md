# wcstack
Web Components Stack

Web Components Stack provides a set of tools for developing applications using Web Components.

## What we prioritize
- Making Web Components easy to use
- Declarative authoring
- Zero configuration
- Low learning curve
- High predictability
- Minimize dependencies as much as possible
- Buildless

## Concept
We aim for "easy to write, easy to read, and hard to break" in daily development while leveraging standard Web Components technologies.
The design minimizes setup and dependencies, making it easy to integrate with minimal effort.

## Recommended for
- Those who want to put Web Components into practical use immediately
- Those who value developer experience over configuration
- Those who want a declarative and consistent API

## What you can expect
- A complete foundation for app development, including routing and layout
- Easy-to-understand naming and structure, making it suitable for team development
