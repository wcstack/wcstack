Fine-grained signals binding a headless <wcs-fetch>

Fine-grained signals drive real DOM with h() and For(). bindNode() adapts the headless <wcs-fetch> node into signals (value / loading / error), and a reactive url input — written back with bindInput — makes the element refetch and abort the in-flight request on every page change, so no stale response can land. Data comes from the shared /api/items server (same endpoint as the other four demos).
