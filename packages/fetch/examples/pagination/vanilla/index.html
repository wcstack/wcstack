Pagination — Vanilla

Hand-rolled state + DOM, <wcs-fetch> via @wc-bindable/core

The framework-free baseline: a plain <wcs-fetch> node does the fetching, and @wc-bindable/core's bind() streams its value / loading / error into a tiny state object that drives hand-built DOM nodes. Setting fetcher.url refetches and aborts the previous request — no AbortController here. It hits the same shared /api/items server as the React, Vue, @wcstack/state and @wcstack/signals demos.
